# Responsive Weather App

## Project Overview
A mobile-first weather application that displays real-time weather data for any location. The app is fully responsive and provides a **smooth user experience** across devices.

This project demonstrates **React component structure, API integration, and responsive frontend design**.

---

## Features
- Search for weather by city.  
- Display current temperature, conditions, and forecasts.  
- Fully responsive layout for mobile and desktop.  

---

## Tech Stack
- React.js  
- HTML5 / CSS3 / Bootstrap  
- JavaScript (ES6+)  
- SheCodes Weather API

---
